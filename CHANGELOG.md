# CHANGELOG

## [6.2.0] - 2026-05-25

### 追加
- **飛行記録の編集機能**
  - 各記録に「✏️ 編集」ボタンを追加
  - モーダル形式で全項目を修正可能
  - 保存するとリストに即時反映

### 法令改正対応
- **2025年12月18日 審査要領改正（施行済み）**
  - 民間資格（JUIDA・DPA等）を活用した申請優遇措置が廃止
  - パイロット情報タブの民間資格欄に警告バナーを追加
- **2026年通常国会 小型無人機等飛行禁止法改正案（閣議決定済み・施行見込み）**
  - 重要施設周辺の飛行禁止エリアが300mから1,000mに拡大予定
  - イエローゾーン内飛行の直罰化予定（6月以下の拘禁刑または50万円以下の罰金）
  - チェックリストの「特定重要施設」項目に改正案の警告を追記
  - 地図凡例に「改正案で1km拡大予定」の注記を追加
  - law_status.json を2025-12-18付に更新

  - 港湾・漁港 / 農地・田畑 / 森林・林道・山岳 / 文化財・史跡・神社仏閣
  - 学校・病院・官公庁 / スポーツ施設・競技場

  - 市区町村役所（財産管理課・土木課・観光課・スポーツ振興課など）

## [5.0.0] - 2025-05-23

### 追加
- **飛行記録タブ（様式１）**
  - 航空法施行規則第236条の84・国土交通省「無人航空機の飛行日誌の取扱要領」準拠
  - 必須記載項目をすべて網羅（飛行年月日・操縦者・離着陸場所・時刻・飛行時間・総飛行時間・特定飛行種別・安全影響事項）
  - 記録はlocalStorageに保存（ブラウザ内完結）
  - **CSV出力**（BOM付UTF-8・Excel対応）
  - **Excel出力（.xlsx）**（SheetJS使用・様式１準拠2シート構成：飛行記録＋サマリー）

- **気象情報（Open-Meteo連携）**
  - APIキー不要・完全無料
  - 飛行地点指定と同時に天気・気温・湿度・風速・瞬間最大風速・視程を取得
  - 風速5m/s超・10m/s超・悪天候・視程不良を自動検知して警告表示
  - 飛行可否判定（✓ 可能 / ⚠ 条件付き / ⛔ 注意）

- **飛行ルート描画**
  - 地図上でルート描画モードON/OFFを切り替え
  - クリックで経由点を追加しポリラインで可視化
  - ルート総距離・全経由点での最寄り空港距離をリアルタイム表示

### 改善
- タブを5つに拡張（飛行チェック・飛行前確認・飛行記録・パイロット情報・チェック履歴）
- リセット時にルート・気象情報もクリア
- 飛行日付のデフォルト値を今日に設定
